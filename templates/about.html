About CookBot

CookBot was my capstone project for the Galvanize Web Development Immersive program. It is a fully decoupled app with access to thousands of recipes. Features include multiple search options and "Cooking Mode", where recipe instructions are read aloud and audio commands are recognized allowing users to easily navigate recipes, hands-free, while cooking.

There is also a Saved Recipe feature that allows users to save recipes they have found to a database. Users can also include notes on the saved recipes.

Built using: IBM Watson Text-to-Speech API, AngularJS, Node.js, Express, PostgreSQL, Knex.js, Sass, Spoonacular API
